== Description ==

This plugin adds support for exporting settings saved from Genesis Simple Sidebars plugin, via the exporter within the Genesis Framework.

== Installation ==

1. Upload the entire `genesis-simple-sidebars-exporter` folder to the `/wp-content/plugins/` directory
1. Ensure Genesis Simple Sidebars plugin is already activated.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. With sidebars already created via `Genesis > Simple Sidebars`, navigate to Genesis Import/Export.
1. Ensure the relevant checkbox is selected and click the Download Export File button.
1. Head to the new site, and import the settings file via the corresponding Import/Export page.
